# 生命树 - 轻社交平台

## 项目简介
生命树是一个轻社交应用，用户可以通过选择人生节点来分享和交流人生感悟。应用以一棵树的形式展示不同的人生主题，用户可以在选定的节点上发布内容、点赞和评论。

## 功能特性

### 1. 用户注册
- 用户名和6位数密码注册
- 用户信息保存在本地存储中

### 2. 生命树界面
- 4个主干代表不同人生主题：成长、事业、情感、兴趣
- 每个主干包含3个可选择的人生节点
- 点击主干显示对应的节点

### 3. 节点选择
- 用户可以选择自己经历过的关键人生节点
- 选中节点后自动跳转到对应社区页面

### 4. 社区互动
- 在选定的节点社区中发布感悟
- 点赞和评论功能
- 实时查看其他用户的分享

### 5. 数据持久化
- 所有数据使用浏览器本地存储
- 用户信息、选择的节点和社区内容都会保存

## 使用说明

1. 访问应用后，输入用户名和6位数密码进行注册
2. 进入主页面，点击任一主干（成长/事业/情感/兴趣）
3. 选择你经历过的节点（点击节点可选择/取消选择）
4. 选择节点后会自动跳转到对应的社区页面
5. 在社区页面可以发布内容、点赞和评论

## 技术栈

- JavaScript (ES6+)
- 本地存储 (localStorage)
